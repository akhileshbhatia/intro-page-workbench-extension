Instructions to use WorkBench Enhancer

  • Github

Save Your Queries

After you write the query and click on the query button, you can see your recently executed query in the sidebar on left hand side.
The queries are listed as per the date that you executed them on.
Also, you can see the time of the day when you had executed the query.

Reuse your queries

When you double click on a query you had executed earlier,it will get populated in the query textbox again
So no need to use notepad to save your queries any more. :)

Search through your queries

You can search through all your queries by simply typing in the search box present at the top

Delete your queries

If you feel like you wrote a query by mistake and you dont need it anymore, you can delete it by using the delete icon corresponding to that query
